# FPGA LED Hourglass Animation (Verilog)

## Overview
This digital design project implements a 16-LED hourglass animation (along with a Knight Rider effect) using Verilog. Developed in Xilinx Vivado, the project demonstrates a strong understanding of hardware description languages, state machines, and clock division techniques.

## Technical Details
* **Language/Tool:** Verilog, Xilinx Vivado
* **Concepts Applied:** * Finite State Machines (FSM) for controlling sequential LED states (e.g., `fullLeft`, `L6R2`).
  * Clock dividers (counters) to slow down the FPGA's base clock for human-visible LED transitions.
  * Behavioral modeling and waveform simulation.
